# Error Log

Record significant bugs encountered during development along with their root cause analysis and solutions.

## Format

Each entry should follow this template:

```
### [Short Title] — YYYY-MM-DD

**Symptom**: What went wrong (error message, unexpected behavior, etc.)

**Root Cause**: Why it happened.

**Solution**: What fixed it (with code snippets or file references if helpful).

**Files Affected**: List of modified files.
```
